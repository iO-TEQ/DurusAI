# TAGS IN DURO CONTROLLER

- Tags represent I/O and internal values (number or text)
- Tags structure is defined in the controller configuration
- Tags can be grouped in folders for better organization
- Tags have configurable properties for persistence, historization, visibility, and editability

Tag Definition Format:

Tags are defined in JSON format with the following structure:

Fields:
- name: The tag’s identifier (e.g., "_NewTag"); used as the JSON key.
- arraydim: Number of elements; 1 means a single value (no array).
- datatype: The tag’s type; "Number" or "Text" for value tags, "Folder" for grouping tags.
- value: The default value; number for "Number", string for "Text". Not present for "Folder".
- config: Object controlling behavior:
  - persistent: Whether value persists across restarts (false in examples).
  - historic: Whether values are historized/logged (false in examples).
  - hidden: Whether tag is hidden from UI (false in examples).
  - editable: Whether tag can be edited (true in examples).
- children: Only for "Folder" datatype; a map of nested tags, each with its own arraydim, datatype, value (for non-folder), and config.

Examples:

1) Here is a number tag with default value 0. 

"_NewTag": {
    "arraydim": 1,
    "datatype": "Number",
    "value": 0,
    "config": {
    "persistent": false,
    "historic": false,
    "hidden": false,
    "editable": true
    }
}
      
2) Here is a text tag with default value "".

"_NewTag_1": {
    "arraydim": 1,
    "datatype": "Text",
    "value": "",
    "config": {
        "persistent": false,
        "historic": false,
        "hidden": false,
        "editable": true
    }
}
      
3) Here is a folder with a text tag in it.

 "_NewFolder": {
    "datatype": "Folder",
    "arraydim": 1,
    "config": {
    "hidden": false,
    "editable": true
    },
    "children": {
    "_NewTag": {
        "arraydim": 1,
        "datatype": "Text",
        "value": "",
        "config": {
            "persistent": false,
            "historic": false,
            "hidden": false,
            "editable": true
        }
    }
    }
}
